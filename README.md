# Flappy Birds (Unity)


Игра, написанная на Unity, в которой игрок с помощью касаний экрана должен контролировать полёт птицы между рядами колон, не задевая их.

Версия Unity: 2018.3.8f1

Последнее обновление: 16.3.2019
